# Nurin Print (EN/DE)

A single-page React + Vite + Tailwind site for a custom apparel & printing store.

## Features
- EN/DE language switcher (in-component i18n)
- Product grid, cart drawer, simple checkout (demo)
- Customizer modal (upload logo, choose technique)
- Pricing cards, FAQ

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy
Deploy the `dist/` folder to any static host (Vercel, Netlify, GitHub Pages).
